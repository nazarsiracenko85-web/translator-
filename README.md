# translator-
Мій перший проект перекладач на CustomTkinter

## -Можливості:
- перекладати прості і складні речення
- перекладати на англіську
- перекладати на німецьку
- перекладати на польску
- перекладати на французьку 
- перекладати на італійську
- перекладати на іспанську

## -Встановлення:
Для запуску необхідно встановити модуль CustomTkinter
```bash
pip install customtkinter
````
